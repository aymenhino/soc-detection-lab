
# Network Baseline Investigation

## Objective
Establish baseline network visibility in the SOC lab environment.

## Lab Environment

Attacker Machine
- Kali Linux

Target Machines
- Windows 10
- Ubuntu Server

Monitoring
- Sysmon
- Wireshark

## Tasks Performed

1. Identify IP addresses of all machines
2. Verify network connectivity
3. Inspect ARP table
4. Capture baseline network communication

## Commands Used

Windows

ipconfig

Linux

ip a
hostname -I

Network testing

ping
arp -a

## Evidence

Screenshots will be stored in the screenshots directory.

## Outcome

Baseline network visibility established for SOC monitoring and future investigations.

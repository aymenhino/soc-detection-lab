# SOC Lab Architecture

This repository documents my personal SOC Analyst lab used to simulate real-world attacks and perform security investigations.

## Lab Components

Attacker Machine
- Kali Linux

Target Machines
- Windows 10
- Ubuntu Server

Monitoring Tools
- Sysmon
- Wireshark

## Objective

Build practical SOC investigation skills including:

- Network traffic analysis
- Log analysis
- Attack detection
- Incident response documentation
